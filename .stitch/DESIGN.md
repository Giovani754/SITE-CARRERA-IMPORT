# CARRERA IMPORTS - Design System Synthesis

## 1. CORE ATMOSPHERE
- **Keywords**: Discret, Premium, Strategic, Trust, Executive, Dark Luxury.
- **Visual Style**: Dark editorial, sophisticated typography, deep shadows, subtle gold/copper accents.
- **User Feel**: "Boutique high-end consultancy", not a dealership.

## 2. COLOR PALETTE
- **Background**: `#050505` (Deep Midnight)
- **Elevated Surface**: `#111111` (Carbide / Obsidian)
- **Primary Accent**: `#D4AF37` (Aesthetic Gold / Copper)
- **Contrast**: `#FFFFFF` (Soft White for headings), `#A1A1AA` (Muted Zinc for body)

## 3. TYPOGRAPHY
- **Headlines**: `Playfair Display` (Serif, elegant, authoritative) - Standard serif fallback.
- **Body**: `Inter` (Modern, clean, legible)
- **Hierarchy**: Large headings with significant letter spacing.

## 4. DESIGN TOKENS
- **Gradients**: Subtle linear gradients (e.g., `#111111` -> `#050505`).
- **Borders**: Hairline borders (`0.5px`) in `#222222` or subtle gold transitions.
- **Shadows**: Soft, diffused glows for gold accents.

## 5. UI/UX KEYWORDS (Stitch Optimized)
- `Atmospheric parallax`
- `Glassmorphism navigation`
- `Smooth scroll reveals`
- `Curated boutique grid`
- `Cinematic imagery`
- `Luxury editorial pacing`
